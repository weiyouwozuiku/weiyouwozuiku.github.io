## 问题描述

任意给定一个32位无符号整数n，求n的二进制表示中1的个数。

## 普通法

原理：移位+计数

```go
func BitCount(a uint8) (num uint8) {
	for num = 0; a != 0; a >>= 1 {
		fmt.Println(a, a&1)
		num += a & 1
	}
	return
}
```

缺点：这样会循环整个byte数组，运行较慢
